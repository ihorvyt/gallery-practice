# Lumina - Фото галерея

Lumina - це сучасна фото галерея на основі Next.js з підтримкою хмарного зберігання (AWS S3), аутентифікацією користувачів та інтуїтивним інтерфейсом для управління фото та папками.

## 🎯 Функціонал

### Основні можливості
- **Реєстрація та авторизація** - Безпечна реєстрація з хешуванням паролів (bcryptjs)
- **Автоматичне збереження сесії** - 7-денний період автологіну через JWT токени в cookies
- **Управління фото**
  - Завантаження фото на AWS S3
  - Перегляд фото у повноекранному режимі
  - Видалення фото
  - Drag & drop для переміщення фото
  
- **Система папок**
  - Створення вкладених папок
  - Перегляд повного шляху навігації (breadcrumb)
  - Клікабельна навігація по папках
  - Видалення папок зі всім вмістом
  - Drag & drop для переміщення фото в папки
  
- **Професійний інтерфейс**
  - Чорновий дизайн з Tailwind CSS
  - Плавні анімації (Motion)
  - Адаптивний дизайн (мобільні, планшети, ПК)
  - Компактна/розгорнута боковина
  - Пошук фото та папок
  - Toast сповіщення

### Технічні особливості
- Next.js 15 (App Router)
- React 19
- Prisma ORM з PostgreSQL
- AWS S3 для зберігання фото
- JWT для аутентифікації
- Signed URLs для безпечного доступу до фото

## 🚀 Встановлення

### Вимоги
- Node.js 18+ 
- npm або yarn
- PostgreSQL база даних
- AWS S3 bucket

### Крок 1: Клонування та встановлення залежностей

```bash
# Встановити залежності
npm install
```

### Крок 2: Налаштування змінних оточення

Створіть файл `.env.local` в кореневій папці проекту:

```env
# База даних
DATABASE_URL="postgresql://user:password@localhost:5432/lumina"

# JWT
JWT_SECRET="ваш-секретний-ключ-мінімум-32-символи"

# AWS S3
AWS_S3_BUCKET="ваш-bucket-name"
AWS_S3_REGION="eu-central-1"
AWS_ACCESS_KEY_ID="ваш-access-key"
AWS_SECRET_ACCESS_KEY="ваш-secret-key"
```

### Крок 3: Налаштування бази даних

```bash
# Встановити Prisma
npx prisma db push

# Або створити міграцію
npx prisma migrate deploy
```

### Крок 4: Запуск в режимі розробки

```bash
npm run dev
```

Проект буде доступний за адресою: `http://localhost:3000`

## 📝 Використання

### Реєстрація
1. Натисніть "Почати" на стартовій сторінці
2. Заповніть форму реєстрації (email та пароль)
3. Вам буде автоматично вважатися логін

### Завантаження фото
1. Натисніть кнопку "Завантажити"
2. Виберіть фото (PNG або JPG, до 10MB)
3. Додайте назву фото (опціонально)
4. Натисніть "Завантажити"

### Управління папками
1. Натисніть "Папка" щоб створити нову папку
2. Введіть назву папки
3. Натисніть на папку щоб відкрити її
4. Используйте breadcrumb для навігації або натисніть на назву папки щоб повернутися

### Drag & Drop
- **Перетягніть фото на папку** щоб перемістити його туди
- **Перетягніть фото на пусту площу** щоб перемістити його на рівень вище

### Автоматичний логін
- Після першого логіну ви залишатиметесь в системі на 7 днів
- При повторному вході на сайт ви автоматично повернетесь в кабінет (якщо усе ще увійшли)

## 🔐 Безпека

- Паролі хешуються з допомогою bcryptjs (10 раундів)
- JWT токени зберігаються в HttpOnly cookies
- Доступ до фото через signed URLs (дійсні 1 годину)
- Користувачі можуть видаляти тільки свої фото та папки

## 🐛 Можливі проблеми

### "Module not found: Can't resolve '@aws-sdk/s3-request-presigner'"
```bash
npm install @aws-sdk/s3-request-presigner
```

### Помилка підключення до PostgreSQL
- Перевірте що PostgreSQL працює
- Перевірте правильність DATABASE_URL в .env.local
- Переконайтесь що база даних існує

### AWS S3 помилки
- Переконайтесь що S3 bucket дійсно існує
- Перевірте IAM дозволи на доступ до S3
- Перевірте що AWS_ACCESS_KEY_ID та AWS_SECRET_ACCESS_KEY правильні
- Переконайтесь що bucket **дозволяє** доступ без ACL (рекомендується для безпеки)
